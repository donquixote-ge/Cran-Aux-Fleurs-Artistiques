# გალაკტიონი

## თავის ქალა არტისტული ყვავილებით

> Le charme inattendu d'un bijou rose\
> et noir,\
> — **Baudelaire**

> La demoiselle bleue aux bords frais\
> de la source,\
> — **Th. Cautier**

> La melancolie des soleils souchants,\
> — **Paul Verlaine**

> ...et les roses trop hautes.\
> **H. de Régnier**
